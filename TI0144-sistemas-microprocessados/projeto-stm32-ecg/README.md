# Projeto Final

Desenvolvimento de um aparelho ECG utilizando STM32.

## Materiais utilizados:

- Microcontrolador STM32 (Greenpill)

- Módulo AD8232 (Sensor cardíaco)

- Pmod R2R

- Osciloscópio
